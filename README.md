# 视频帧提取器

从视频中提取封面和缩略图的自动工具，配合EMBY之类的使用。

功能：

1。支持JPG或PNG格式抓取

2。支持文件夹及文件拖入自动获取视频

3。支持首帧提取

4。支持指定帧提取

5。支持随机帧多帧提取，并可设置提取时间范围（按比例设置）

6。可自行设置抓取图片保存地址。

7。支持操作记录及设置保存。

8。支持抓取后选图自动在视频对应地址生成封面图和缩略图，并自动对图片进行裁剪适合EMBY用。

其它功能自己摸索吧。
